
# git指令
git status 检查/查看哪些文件被修改或者被新增  
git add [file] 将要提交的文件添加到修改文件中  
git add . Or -A 将所有修改过的文件添加到暂存区  
git commit -m ""提交暂存区的代码，并附上描述性的提交信息  
git checkout -b [分支名称]   创建一个新的分支，并立即切换到这个心分支上  
git branch 验证您当前所在的分支  
git push -u origin [分支名称] 首次将本地分支推送到名为origin的远程仓库，-u参数会简历本地分支和远程分支的关联，之后只需要使用git push  
git push 在第一次推送之后，只需使用此命令即可
